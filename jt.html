Routing Protocol

  1. 정적 라우팅 static Route

    2. 정적 경로 설정이 동적인 네트워크 상황 변화에 자동으로 적응하지 못하고 라우팅 테이블을 수동으로 설정해야 하므로 많이 사용하지 않을 것 같지만 실제로 폭넓게 사용된다.

  2. 동적 라우팅 Dynamic Route

    3. 동적 경로 설정은 동적 라우팅 프로토콜에 의해 자동으로 네트워크 탐색 및 등록과정을 수행하고 이에 따라 라우팅 테이블을 완성한다.

    종류 : EIGRP(distance-vector방식), OSPF(link-state방식)
